05 / 模型说明

模型机制与规则

基本概念

不回溯历史 = 每季度滚动校准（无未来信息），回测与评估用

回溯历史 = 全样本拟合（含未来信息），仅展示用

情绪指数 = 模型原始预测值 raw_total（不做转换映射）

两阶段回归结构

转换y = -(yield - μ) / σ（将10Y收益率标准化并反转方向，数值越大 = 债市越强）

阶段一：各类内部因子 → z_score加权 → 大类得分

阶段二：四大类得分 + 截距 → 情绪指数（raw_total）

原始y = 10Y国债收益率

贡献分解公式

大类贡献 = β₂_c × (Σβ₁ᵢzᵢ + 截距_c)

类内因子贡献 = z_score × β₁ × β₂_c

残差 = 截距₂

情绪指数 = 交易热度贡献 + 机构行为贡献 + 品种利差&IRS预期贡献 + 宏观资产比价贡献 + 残差
