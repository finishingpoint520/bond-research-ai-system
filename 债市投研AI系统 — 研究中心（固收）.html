01 / 系统全景

两条工作流，一个目标

就像飞机的两个引擎：量化模型是传感器——告诉你当前的速度/高度/姿态；Agent审计系统是驾驶舱——把所有信息整合成可执行的决策。

债市五因子多Agent策略审计系统

定性框架的 Agent 化  ·  12个Agent协作

  • 5个因子Agent 分别对基本面/政策面/流动性/市场情绪/机构行为进行四维打分
  • Regime Agent 识别市场状态（宽松扩散/收紧传导/博弈震荡/拥挤反转/风险逃离）
  • Narrative Agent 提取当前主导叙事 Top 3
  • Crowding Agent 评估交易拥挤度，影响赔率和仓位
  • Pricing-Odds Agent 区分估值赔率和交易赔率
  • Case Retrieval Agent 检索历史相似案例，提供胜率参考
  • 策略生成 + Verifier审计 合成策略 → 反证审计 → 可执行建议
输出：结构化策略审计报告（评分表 / 冲突表 / 情景树 / 证伪点 / 审计意见）

债市AI量化模型系统

定量信号的体系化  ·  三大模型矩阵

  • 宏观高频景气指数 (v2) 追踪经济增长、通胀、融资需求等维度的综合景气判断
  • 流动性量化模型 (迭代中) 建模DR007/存单等资金利率的中枢与边际变化，日度信号+强度值
  • 机构情绪择时指数 (已上线) 从机构行为和市场情绪指标中提取对10Y有择时意义的因子，每日邮件推送
  • 机构情绪同步指数 (拟合中) 衡量当前市场整体交易热度（过冷/中性/过热），与10Y高度同步，辅助仓位管理
  • 跨资产验证体系 整合债券/股票/商品/汇率周度信号，形成跨市场验证闭环
输出：量化信号序列 + 可视化图表（方向 / 强度 / 分位数 / 回测）

03 / 范式之变

从"人写报告"到"人审系统"

研究员的价值从"写出判断"升级为"审核系统生成的判断，并挑战它"——这是质的飞跃，不是量的提升。

传统投研流程

以人为中心的线性生产

  • 研究员收集数据 （约 2h）
  • 研究员分析判断 （约 3h）
  • 研究员撰写报告 （约 2h）
  • 研究员交叉验证 （约 1h）
约 8 小时 / 人  ·  覆盖 3 个维度  ·  无系统反证机制

AI 增强后的流程

以系统为中心，人负责审核与挑战

  • Agent 自动取数 + 四维打分 （自动）
  • 12个Agent并行分析 + 交叉验证 （自动）
  • 研究员审核 + 修正 （约 30min，核心价值）
  • 自动反证审计 + 可执行性检查 （自动）
45分钟 AI + 30分钟人  ·  覆盖 5因子 × 4维评分  ·  内置证伪 + 冲突审计
