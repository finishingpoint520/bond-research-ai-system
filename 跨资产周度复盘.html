MODULE 2 · DIVERGENCE SCAN
背离扫描与因果甄别

背离一：STAR50 +10% vs 广度崩溃至4/31 — 最危险的背离 趋势型

【现象】
STAR50 +10.0%、科创50周四单日涨超8%。但仅4/31个行业上涨——有色金属+4.6%、传媒+3.5%、计算机+2.1%、电子+1.2%。其余27个行业全部下跌，银行-7.0%、综合-8.2%。科创50与银行周收益差达+17.1pp——超越P2峰值。
【隐含逻辑】
AI产业叙事（GPT-5.6 + Meta否认算力过剩 + 美光$2500亿）点燃了科技股——但资金来源是存量调仓而非增量入场。银行-7%、红利-1.2%、微盘-5.0%是科技暴涨的"血包"。这不是"市场普涨"，这不是"risk-on全面扩散"——这是P1-P2虹吸结构的精确重演。涨幅集中度（前4行业占总涨幅比例）回到极端水平。
【成因甄别】
趋势型背离。R3/R4/R7同步从🟢/🔵退回⚠虹吸预警——这不是"一日游"的噪声。但与P1-P2的关键区别：花旗中国从-48.5改善至-28.1 + ETF资金改善而非恶化 + Brent反弹至$76——宏观背景不同。本轮虹吸有基本面边际支撑，P1-P2没有。
【失效条件】
若下周广度恢复至>15/31 + ETF继续保持净申购 → 虹吸可能"软着陆"为普涨——本轮与P1-P2不同。若广度进一步恶化至<3/31 + ETF再次转为净赎回 → 虹吸破裂风险急剧上升,科创50单周-5%+的概率大幅增加。

背离二：美伊冲突升级 vs VIX 15.84 — 地缘风险完全未被定价 事件型

【现象】
美国打击90个伊朗军事目标，伊朗袭击美在中东军事基地，伊朗24小时运出1100万桶原油。Brent单周+6.3%至$76——但VIX仅15.84，处于极度平静水平。期权市场完全没有在定价一场可能的地区战争。
【隐含逻辑】
市场当前的定价假设是"冲突可控"——巴基斯坦仍在斡旋，双方有重返谈判的渠道。2019年9月沙特阿美遭袭（Brent单日+14.6%）和2020年1月苏莱曼尼事件（Brent+4%）都曾被市场快速消化——地缘溢价通常在1-2周内消退。但如果霍尔木兹海峡通行受阻（全球约20%石油经此通过），Brent可能跳升至$90+——这是VIX 15.84完全没有定价的场景。
【成因甄别】
事件型背离。VIX的平静有历史经验支撑——过去的美伊冲突中，如果没有实际影响石油运输，市场反应都是短暂的。R8从⚫休眠升级为🟡预警——不是因为冲突本身,而是因为VIX未定价意味着如果冲突升级,风险重定价将是跃迁式的。
【失效条件】
若美伊重回谈判→Brent回吐地缘溢价→背离消失。若冲突升级至霍尔木兹海峡通行受阻→Brent单日+10%+→VIX突破22→R8🟡→⚠升级→地缘风险从"未定价"切换为"紧急重定价"。

背离三：ETF资金全面改善 vs 广度崩溃 — 谁在买？买了什么？ 趋势型

【现象】
159915（创业板ETF）+63,500万份——近两个月最大单周流入。588000（科创50ETF）+11,100翻正。510300赎回改善70%。但广度崩溃至4/31——ETF资金在涌入,但绝大多数个股在下跌。
【隐含逻辑】
ETF资金主要通过权重股流入——科创50和创业板的权重股（半导体、AI、新能源龙头）吸收了大部分资金，而这些权重股恰好是本周领涨的少数行业。ETF的被动配置属性意味着资金流入天然偏向权重股——在行业广度本已脆弱的时期,ETF大额流入反而加剧了"指数涨+个股跌"的结构性扭曲。
【成因甄别】
趋势型背离。这是本轮周期首次出现"ETF大额申购+广度崩溃"的组合——历史上这种组合通常出现在市场从底部反弹的早期（如2022年11月reopen trade），而非反弹的延续期。当前更接近前者——花旗中国从-48.5改善至-28.1,宏观底部可能在确认中。
【失效条件】
若下周广度恢复至>15/31 → ETF资金从权重股扩散至全市场 → "底部反弹"确认。若ETF申购放缓+广度维持<5/31 → ETF申购的"托市效应"消退 → 指数可能跟随个股下跌。

MODULE 4 · EXPECTATION GAPS
潜在预期差提醒

叙事差1：「AI交易重燃=调整结束」vs 本轮虹吸结构与前一轮完全相同

市场共识：GPT-5.6+Meta否认算力过剩 → AI叙事回归 → STAR50暴涨=调整确认结束。底层裂缝：4/31广度、银行-7%、红利-1.2%、成长vs价值周收益差+1.6pp——这个结构两周前刚引发科创50单日-7.7%的暴跌。本轮与上轮唯一区别是宏观背景和资金面更好——但"更好的背景"能否阻止虹吸破裂？历史经验显示：花旗意外指数的改善需要3-6个月才能传导至行业广度的修复——当前花旗才改善5周。

叙事差2：「美伊冲突可控=油价已定价」vs VIX未定价=跃迁风险

市场共识：巴基斯坦仍在斡旋→双方有重返谈判渠道→冲突不会升级至影响石油运输→Brent+6.3%已充分定价。底层裂缝：VIX 15.84意味着期权市场给"冲突升级至霍尔木兹受阻"的概率接近零——这不是"定价了可控"，是"没有定价失控"。一旦巴基斯坦斡旋失败+美军扩大打击范围，Brent可能从$76跳升至$90+，跨资产波动率可能出现跃迁式跳升。

尾部风险

1. 霍尔木兹海峡通行受阻 → Brent跳升至$90+。若美伊冲突波及海峡通行——全球约20%石油经此通过。R8直接从🟡升级为⚠激活。概率低（巴基斯坦斡旋仍在进行），但若发生则是系统性冲击。

2. 二次虹吸破裂。若下周广度继续恶化+ETF申购放缓 → STAR50可能重演两周前-7.7%的单日暴跌。本轮ETF资金改善提供了缓冲——但如果ETF申购也出现逆转（159915从+63,500转为净赎回），缓冲消失。

3. CPI/PPI双降→通缩预期自我强化。CPI+1.0%、PPI+4.1%环比双降——如果这一趋势持续,市场可能从"通胀缓解利好"切换到"通缩担忧利空"。届时中债10Y可能下破1.70%,而A股将面临"物价下跌→企业盈利恶化"的重新定价。
