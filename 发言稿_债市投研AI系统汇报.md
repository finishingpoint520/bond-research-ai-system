# 债市投研AI系统建设汇报发言稿

> 发言时长：约 10 分钟（约 2500 字）
> 汇报人：研究中心（固收）
> 对象：公司领导、投资总监
> 数据截止：2026年7月11日当周

---

## 一、我们在做什么（1 分钟）

各位领导，今天汇报的是一项仍在进行中的工作——过去一年研究中心（固收）尝试把 AI 引入债券投研的底层流程，目前形成了两条并行的工作线。

**第一条线**，是定性框架的 Agent 化——"债市五因子多 Agent 策略审计系统"。简单说，就是把研究员日常判断市场时会看的五个维度——基本面、政策面、流动性、市场情绪、机构行为——分别交给独立的 AI Agent 去分析，再由一个审计 Agent 对所有结论进行系统性挑错。

**第二条线**，是定量信号的体系化——"债市 AI 量化模型系统"。包含了宏观景气、流动性、机构情绪等方向的量化模型，输出的是可以每日追踪、可以回测验证的数值信号。

两条线共享同一套数据底座，逻辑上是上下游关系：量化模型提供客观信号，Agent 审计系统把信号纳入完整的分析框架，最终由人工复核形成策略决策。我们的目标不是"用 AI 写周报"，而是把研究员的核心价值从执行分析升级为审核判断。

---

## 二、整体架构（1 分钟）

五因子 Agent 系统目前接入了三个因子——基本面、流动性、机构行为，政策因子和市场定价因子待嵌入。每个因子输出四维评分：当前状态、边际变化、预期差和置信度。然后由总控 Agent 汇总，识别因子间的冲突，生成策略建议，最后交给审计 Agent 逐一验证数据是否可靠、逻辑是否自洽、是否存在被忽视的反证。

量化模型系统则由三个引擎构成：宏观高频景气指数追踪经济运行的方向和温度，流动性模型聚焦资金价格中枢的边际变化，机构情绪指数体系捕捉市场参与者的行为信号。此外，银行资负 Skill 和跨资产跟踪周报分别在微观机构行为和宏观跨市场层面提供补充验证。

---

## 三、八个已交付模块（6 分钟）

下面逐一介绍每个模块的设计思路、本周实际运行结论以及当前面临的困难。

### 1. 五因子多 Agent 策略审计系统

这是整个定性分析体系的骨架。设计思路是朴素的：不信任任何单一模型给出的结论，所以让多个 Agent 互相校验。

**本周结论**：三因子全部落入中性区间——基本面、流动性、机构行为的净分均小于 0.4，置信度均为 +1。市场处于方向选择期。机构行为边际转正提供弱支撑，但逆回购余额降至 620 亿（10% 分位），央行有回收流动性的意图。当前信号强度不足以支撑方向性判断，10 年国债预计在 1.72% 到 1.78% 之间继续震荡。

**当前困难**：政策因子和市场定价因子尚未接入，这是系统从"三因子"升级到"五因子"的关键一步，依赖指标字典与 IT 的数据管道对接。

### 2. 政策-舆情-预期差 Skill（v3）

这个模块要解决的，是如何把央行表态、政治局会议措辞、求是网署名文章这些非结构化信息，量化成可比较的方向性信号。

**本周结论**：政策面继续偏多。预期差从上周的 +0.901 收敛到本周的 +0.209，仍为正。含义是：政策仍领先市场半步，利率下行空间尚未被完全定价，但交易赔率不如上周宽。本周最重要的边际变化来自求是网——三篇文章密集发布，"适度宽松"被党的理论刊物正式确认，"双支柱"框架进入制度化表述。这消除了 Q2 货政委例会"宏观审慎评估债市"可能被市场误读为"收紧信号"的风险。一句话：双支柱是制度建设，不是打压。

**当前困难**：政策语义词典的边界清晰度还需要提升。比如"稳健偏松"和"灵活适度"之间的量化差异，目前的标注精度还有提升空间。

### 3. 宏观高频景气指数（v2）

追踪经济增长、通胀、外需等维度的周度和月度高频数据，合成一个综合景气判断。

**本周结论**：宏观景气总指数 Z-score 为 -0.39，处于五年百分位的 41.0%，中性。分项中，外需偏强（67.2% 分位），工业生产和内需均处于中性区间。总体而言，宏观基本面未对债市形成明确的方向性驱动。

**当前困难**：新指标口径切换后，历史可比数据的时间序列长度不够，需要更长的运行时间才能形成稳定的统计推断。

### 4. 流动性量化模型

聚焦资金利率对债市的影响。目前正在从月度信号升级为日度信号，并增加了信号强度值。

**本周结论**：DR007 相对 OMO 处于 17% 分位低位，流动性本身偏松；但逆回购余额大幅降至 620 亿，处于 10% 分位，说明央行在主动回收流动性而非投放。这种"松现实、紧预期"的组合，对债市形成中性偏谨慎的判断。

**当前困难**：新模型的 Y 值设定方案和预测周期还在与 IT 同事讨论，候选指标测试需要时间。日度信号的稳定性验证也刚开始。

### 5. 机构情绪择时指数

设计目标是：从 13 个底层因子中提取对 10 年国债有择时意义的信号。它追求的是领先性——预测未来 20 个交易日的方向。已上线，每日邮件自动推送。

**本周结论**：机构情绪指数目前处于阈值之间的中性区域，未触发明确的看多或看空信号。这与五因子系统的判断一致——市场处于方向选择期。

**当前困难**：触发率与胜率之间存在天然 tension——提高阈值可以提升胜率但会降低信号频率，需要在实际运行中根据投资需求动态平衡。

### 6. 机构情绪同步指数

与择时指数互补。择时指数追求领先性，同步指数追求一致性。

**本周结论**：同步指数基于两阶段回归模型，将四大类因子（交易热度、机构行为、品种利差与 IRS 预期、宏观资产比价）分解为每日贡献。目前指数显示市场整体交易热度处于中性水平，未出现过热或过冷信号。当同步指数与 10 年国债方向高度一致时，拥挤度上升意味着进一步上涨的动能减弱，市场对利空会更敏感。

**当前困难**：主导因子的提取和拐点识别策略还在探索中，目前只有一个初步拟合版本。需要积累更多样本后才能稳定输出拥挤度预警。

### 7. 银行资负 Skill

这个模块的切入角度与前面不同。它站在银行司库的视角——代入流动性安全、利率风险管理和利润考核三层目标——从微观主体的决策行为反推宏观资金面的走势。

**本周结论**：核心传导链是"银行资负 → 大行融出 → 资金价格 → DR001 方向"。本周大行融出规模的变化正在被纳入监测框架。这个模块不是独立运行的，它的输出将直接服务于流动性因子 Agent 的 state_score 和 marginal_score 判断。

**当前困难**：框架初步形成，但数据链路——特别是大行融出行为的日度数据——还需要与 IT 侧完成对接。

### 8. 跨资产跟踪周报

最后一个模块，解决债券研究容易"只盯债市、不看全局"的问题。

**本周结论**：全球 risk-on 仍在——纳指上涨 1.7%，高收益债利差收窄 5 个基点至 2.70%，VIX 降至 15.0。但利率和油价开始构成约束——美债 10 年上行 7 个基点至 4.56%，布伦特原油上涨 5.4% 至 76 美元。对国内债券的含义是：A 股走弱提供了避险逻辑支撑，但全球利率中枢上移限制了国内利率的下行空间。这组矛盾信号与五因子系统的中性判断形成交叉验证。

**当前困难**：跨资产的信号融合权重——比如美股下跌和 A 股走弱对国内债券的传导系数不同——还需要积累更多样本后才能稳定。初步版本已形成，目前处于周度手工运行阶段。

---

## 四、总结与下一步（1 分钟）

总结三句话。

**第一句**：八个模块中，机构情绪择时指数已进入每日运行状态，政策 Skill 完成了两年回测并迭代至第三版，五因子系统已接入三因子并输出了实际判断。总体架构设计已经完成。

**第二句**：当前核心瓶颈不在方法论层面，而在工程化落地和数据管道层面。五个因子中有两个（政策、市场定价）尚未接入主系统，银行资负和跨资产模块目前还是手工运行，需要 IT 侧支持完成标准化对接。

**第三句**：我们不追求一次性的 Demo，而是建设一个可以每周产出、每月迭代的生产系统。按计划，七月实现五因子系统首次端到端运行，九月完成量化模型的综合信号融合，年底争取让策略信号进入实战辅助决策。

这个方向没有捷径，但经过过去一年的验证，我们确认这是对的。谢谢。
