Skill — 资金面/流动性视角

从银行资产负债状态出发，研判债市资金面（狭义流动性）走势。核心方法论：站在银行司库（Treasury）视角，代入"流动性安全 → 利率风险管理 → 利润考核"三层目标，微观主体决策推演。

01 / 概览

Skill 要解决什么

核心聚焦银行资产负债状态变化对债市资金面/流动性的传导，每一步分析直指 DR001 方向与 R001-DR001 利差变化。

银行资负 → 大行融出 + CD市场 + 结构性摩擦 + 监管脉冲 → DR001 + R-DR利差
分析领域

资金面/流动性研判

DR001/R001/R-DR利差方向判断、大行融出行为、存单量价信号、银行资负结构性变化、季末监管脉冲等。

核心方法

银行司库视角推演

站在银行 Treasury 视角，代入流动性安全→利率风险管理→利润考核三层目标，每一轮资金面变化都追溯到司库具体行为选择。

分析落点

DR001 + R-DR利差

所有分析最终归因到 DR001 中枢方向和 R001-DR001 利差变化，直接服务于杠杆策略、CD配置、套息交易等债市决策。

04 / 分析框架

六步法：每步落点 DR001 / R-DR利差

第一步

大行融出研判 — 隔夜资金的"水源"

决定 DR001 的最核心因子

核心问题：大行有多少可融出资金？愿不愿意往外借？大行存贷增速差 → 融出规模 → DR001 中枢方向，是大行资负状态最直接的映射。

第二步

CD市场信号 — 负债压力的高频窗口

CD量价变化领先 DR001 方向

CD是观测银行负债压力的最佳高频指标。CD提价发行 → 银行负债压力上升 → 大行融出意愿下降 → DR001中枢上移。CD是否有效突破MLF（1.50%）是关键信号。

第三步

结构性摩擦研判 — 超储在谁手里？

总量超储充裕 ≠ DR001低

核心认知：关键是超储分布在能融出的机构手中。R-DR利差的核心驱动：大行负债压力↑ → 减少隔夜融出 → R001-DR001走阔（分层加剧）；季末MPA+LCR → 大行融出收缩 → R001-DR001脉冲式走阔（三重驱动：MPA考核+同融占比上限33%+货基赎回负反馈）；非银资金充裕 → 配置CD → 间接压低短端 → R001-DR001收窄。

第四步

央行态度研判 — 水龙头开多大？

DR001是检验央行态度变化的最即时窗口
央行操作组合 态度信号 对 DR001 的含义
持续净投放（OMO+MLF+买断式） 呵护 DR001中枢下移
归零操作（逆回购量为0） 认为资金过松，回归中性 短期→DR001触底后缓慢回升
缩量续做+净回笼 偏紧 DR001中枢上移
重启国债买入 补充长期流动性 超储长效改善→DR001中枢下移
降准 强烈宽松 DR001中枢显著下移

DR001最直接反映央行每日OMO操作的态度——当OMO净投放时大行隔夜融出最敏感，DR001比DR007更快反应。

第五步

资金面"自发性宽松/收紧"的条件判断

自发性宽松

需同时满足4个信号

参考2022年、2026H1情形：大行融出规模持续高位 + CD量价双稳（未突破MLF） + 超储分布均衡（大行占比高） + 央行态度呵护（持续净投放）

自发性收紧

需警惕4个信号

大行融出规模骤降 + CD提价放量（接近或突破MLF） + 超储向中小行转移（结构性摩擦加大） + 央行态度边际转紧（净回笼/归零操作）

第六步

综合研判 → DR001 + R-DR利差

整合前五步信息，形成完整判断
DR001中枢

低于 / 贴近 / 高于
政策利率 1.40%

判断当前DR001中枢相对OMO利率的位置及短期方向。

R-DR利差

低位 / 正常 / 走阔
/ 明显分层

判断R001-DR001分层状态及方向，反映非银资金面的结构性紧张程度。

策略含义

杠杆 / CD / 套息
策略建议

基于DR001与R-DR利差判断，输出杠杆策略友好度、CD配置性价比、套息交易环境评估。

06 / Skill 输出格式

标准输出结构

资金面分析按以下8步结构输出，每一步都归因到 DR001 / R-DR利差。

  1. 总体判断 — DR001中枢 + 方向 + 一句话定性；R001-DR001分层状态
  2. 大行融出研判 → DR001方向 — 存贷增速差、融出规模、冗余存款变化
  3. CD市场信号 → DR001压力 — 量价组合、大行发行占比
  4. 结构性摩擦 → R001-DR001利差 — 超储分布、揽储分化、货币传导摩擦
  5. 央行态度 → DR001政策环境 — 态度指数、操作组合
  6. 监管脉冲时点 — 距季末/税期距离，预期对DR001和R001-DR001的短期冲击
  7. 综合结论 + 对杠杆/套息策略的含义
  8. 风险提示
