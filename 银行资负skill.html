银行资负分析专家
Skill — 资金面/流动性视角

从银行资产负债状态出发，研判债市资金面（狭义流动性）走势。站在银行司库（Treasury）视角，代入"流动性安全 → 利率风险管理 → 利润考核"三层目标，从微观主体决策推演宏观利率走势。

01 / 概览

Skill 要解决什么

核心聚焦银行资产负债状态变化对债市资金面/流动性的传导。每一步分析直指 DR001 方向与 R001-DR001 利差变化。

银行资负 → 大行融出 + CD市场 + 结构性摩擦 + 监管脉冲 → DR001 + R-DR利差

04 / 核心传导链

从银行资负到资金面价格的五条传导路径

银行资负状态
大行存贷增速差 → 大行融出规模 → DR001 中枢
存款搬家 / CD到期 → CD 量价信号 → 资金面压力
大行/中小行超储分布 → 结构性摩擦 → R-DR 利差

05 / 分析框架

六步法：每步落点 DR001 / R-DR利差

第一步

大行融出研判 — 隔夜资金的"水源"

决定 DR001 的最核心因子

核心问题：大行有多少可融出资金？愿不愿意往外借？

研判维度包括：大行存贷增速差、超储率、大行相对揽储优势、央行态度指数、1Y股份行NCD利率。

第二步

CD市场信号 — 负债压力的高频窗口

CD量价变化领先 DR001 方向
CD信号负债端含义对 DR001 / R-DR 的含义
量升+价升补负债需求+成本同步上升DR001倾向上行
量降+价降负债压力缓解DR001倾向下行
大行CD发行占比↑大行也开始缺负债大行隔夜融出将收缩 → DR001↑
CD净融资持续为正存量负债缺口在扩张DR001承压信号持续
第三步

结构性摩擦研判 — 超储在谁手里？

总量超储充裕 ≠ DR001低，关键是超储分布在能融出的机构手中
结构性因子传导方向对 DR001 / R-DR 的含义
揽储能力分化大行揽储强→超储向大行集中→大行隔夜融出多DR001↓
信贷增长分化大行信贷强→超储消耗多→大行隔夜融出减DR001↑
货币工具传导摩擦中小行和非银滞后R-DR走阔
流动性监管（LCR等）季末银行减少融出→非银承压R-DR脉冲走阔

R-DR利差（资金分层）的核心驱动：
大行负债压力↑ → 减少隔夜融出 → R001-DR001 走阔
季末MPA+LCR → 大行融出收缩 → R001-DR001 脉冲式走阔
非银资金充裕 → 配置CD → 间接压低短端 → R001-DR001 收窄

第四步

央行态度研判 — 水龙头开多大？

DR001最直接反映央行每日OMO操作的态度——当OMO净投放时大行隔夜融出最敏感。DR001是检验央行态度变化的最即时窗口，比DR007更快反应。

第五步

资金面"自发性宽松/收紧"的条件判断

自发性宽松

需同时满足4个信号

参考2022年、2026H1情形：大行融出规模持续高位 + CD量价双稳 + 超储分布均衡 + 央行态度呵护

自发性收紧

需警惕4个信号

大行融出规模骤降 + CD提价放量 + 超储向中小行转移 + 央行态度边际转紧

第六步

综合研判 → DR001 + R-DR利差

整合前五步信息，形成完整判断
DR001中枢

低于 / 贴近 / 高于
政策利率 1.40%

判断当前DR001中枢相对OMO利率的位置及短期方向。

R-DR利差

低位 / 正常 / 走阔
/ 明显分层

判断R001-DR001分层状态及方向，反映非银资金面的结构性紧张程度。

策略含义

杠杆 / CD / 套息
策略建议

基于DR001与R-DR利差判断，输出杠杆策略友好度、CD配置性价比、套息交易环境评估。

06 / 分情景分析

三种典型情景速查

情景 A

司库流动性"极度宽松" → 回归中性

特征
DR001 < 1.25%，MDS/MLF/CMLF全部倒挂
司库行为
缩表（减少净融出 + 退出MDS/MLF）
DR001路径
1.20% → 1.36% → 1.40%（缓慢回升）
关键信号
央行开始"纠偏"（OMO归零 / 金融时报喊话）
情景 B

司库面临"临时性冲击"

特征
大行净融出单日骤降近万亿
触发
MDS中标不足 或 信贷窗口指导
司库行为
应激性缩表 → 后续重新摆布头寸 → 渐进回归
DR001路径
骤升 → 次日起渐进回落
关键信号
是否为"申报量 > 中标量"
情景 C

司库资负缺口承压（结构性）

特征
存款外生流失 + CD集中到期 + 回购利率上行
司库行为
提价发CD → 提价吸同业活期 → 划定"司库预算线"控成本
DR001路径
中枢上移至 1.40—1.45%
关键信号
CD是否有效突破MLF（1.50%）
